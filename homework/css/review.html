CSS syntax ประกอบด้วยอะไรบ้าง 1.Selector >> การเลือกElement
2.Property >> คุณสมบัติของElementที่เราเลือก เช่น สี
3.Value >> ค่าของคุณสมบัติ เช่นสีแดง
การเขียน css มีกี่แบบ มีทั้งหมด 3 แบบ
1.lnline การเขียนไว้ในAttribute style ภายในElementที่เราจะใช้css
2.External การเขียนไฟท์ css แยกแล้วlink เข้ามาใน Tag line ในส่วนของ header
3.Embed การสร้าง css ในfile HTML ภายใน style ในส่งของ Hardox
selector มีกี่แบบ มี 3 แบบ
1.กำหนดด้วยชื่อของElement เช่น div,p,h1,img
2.กำหนดด้วยชื่อ class ของ Element โดยใช้ จุด(.)หน้าชื่อclass เช่น .abs,div.x1,p.web1
3.กำหนดด้วยชื่อ id ของ Element โดยใช้ จุด(#)หน้าชื่อid เช่น #abs,div#x1,p#web1
combinator มีกี่แบบอะไรบ้าง มี 4 แบบ
1.ใช้เครื่องหมาย > จะทำงานกับElement ชั้นแรก(ชั้นลูก)และไม่ทำงานกับElement(หลาน-เหลน) เช่น div > p,div > div ,span > a
2.ใช้เครื่องหมาย วรรณวะ(ช่องว่าง) จะทำงานกับทุกElementและทุกระบบชั้น(ลูก-หลาน-เหลน) ของ elemant ที่เรากำหมดหน้าหน้าช่องว่าง เช่น div p, div h3,span a
3.ใช้เครื่องหมาย , จะทำงานกับทุกElementที่ถูกคั่นโดย "," นี้ เช่น h1,h2,h3 ,div#co,div.solo
4.ใช้เครื่องหมาย จะทำงานกับElementทุกตัวที่อยู่ถัดออกไปและไม่ทำงานกับElementที่อยู่ในชั่นลูกหรือตำกว่านั้น เช่น h1
