property และ value คืออะไรแตกต่างกับอย่างไร property คือ คุณสมบัติของelement เช่น กว้าง,ยาว,สี
ส่วน value คือกำหนดค่าชองคุณสมบัติ เช่น สีแดง,100px
Property ที่สอนใน precame มีกี่กลุ่มอะไรบ้าง มี 5 กลุ่ม
Property for content>>กลุ่มที่ใช้กับเนื่อหาภายใน Element เช่น color,size,width
Property for border>>กลุ่มที่ใช้กับเส้นของของ elemant เช่น border
Property for padding>>กลุ่มที่ใช้กับpadding(ระยะห่างระหว่างเนื่อหากับของelement) เช่น padding-left,padding-right
Property for margin>>กลุ่มที่ใช้ระยะห่างระหว่าง Element เช่น margin-right,margin-left
Property for Element>>กลุ่มที่ใช้กับElement(ทั้งหมดของElement) เช่น display,height,position
value unitมีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร มี 2 ประเภทคือ Absolute lenghts(ไม่สัมพันธ์) กับ Relative lenghts(สมพันธ์)
Absoluete lenghts Relative lenghts

mm >> มิลลิเมตร ex >> เท่ากับความสูงของfont element
in >> นิ้ว ch >> ความกว้างของตัวหนังสือใน parent element
px >> พิกเซล 1/96 นิ้ว rem >> เท่าขนาดfontของ root element
pt >> พอยท์ 1/72 in vh >> %ความสูงของหน้าจอ
pc >> พิคาล 1/12 นิ้ว vw >> %ความกว้างของหน้าจอ
