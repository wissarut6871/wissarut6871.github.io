คำถาม คำตอบ
value ใน display property มีอะไรบ้าง มี 5 แบบ
1.inline(ตามcontent)
-ข้อความไม่ขึ้นบรรทัดใหม่ จะขึ้นเมื่อความยาวของข้อความเต็มบรรทัด
-ความกว้าง,ความยาวจะแปรผันตามขนาดของข้อความ
-คุณสมบัติ width,height,margin,padding,ฯไม่มีผลกับdisplayแบบinline
2.block(กล่องข้อความ)
-ขึ้นบรรทัดใหม่ ถึงแม้จะข้อความจะยาวไม่เต็มแถวก็ตาม
-คุณสมบัติ width,height,margin,padding,ฯมีผลกับblock
3.inline-block
-ทุกอย่างเหมื่อนกับdisplay:inlineแต่สามารถปรับแต่งขนาดได้(คุณสมบัติ width,height,margin,padding,ฯมีผล)
4.none(ไม่แสดง)
-เมื่อกำหนดแบบ none elementนั้นจะไม่แสดงบนหน้าจอ
5.grid(ตาราง)
-แสดงผลแบบตาราง,กำหนดคอลัมพ์,แถว,สามารถผสานเซลล์ได้,มีรูปแบบคงที่
6.flex(ไหล)
-เรียงelementไปตามแถว เมื่อเต็มแถวก็ไหลไปแถวต่อไป
สามารถกำหมดทิศทางการไหลของelement ได้
grid และ flex แตงต่างกันอย่างไร
